Reflection

This is the first time I try to build a website with more than 5 pages. The first stage is to design the style and home page of the website, source log page and feedback page and so on.

In the process of programming, I found that in order to keep the head style of each sub page consistent, a CSS file must be set. This file contains the styles of elements common to each subweb page. In this way, when I need to create a new HTML file, I just need to call the parameters in the CSS file to simply reproduce the same effect.

But at the beginning, as long as a new page is added, I will add the new page link to the navigation bar. As a result, the position of the navigation bar is not enough to add a new page, so I changed the design and put the new page into the body of the existing page in the form of hyperlink to reduce the pressure of the navigation bar.
